# 二进制相关

1. 判断一个数是不是二的倍数，即判断该数二进制末位是不是 0：

   a % 2 == 0 或者 a & 0x0001 == 0。

2. 十进制转二进制

   ```C++
   #include<stdio.h>
   int main()
   {
       int x,i;
       scanf("%d",&x);
       for(i=5;i>=0;i--)
           printf("%d",x>>i&1);
       printf("\n");
       return 0;
   }
   ```
   
3. 求某数除以最大的2的次方数的余数

   a&-a（比如10，结果就是2，但是如果是8的话，结果是8）
