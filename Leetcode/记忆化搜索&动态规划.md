# 记忆化搜索
记忆化是为了防止在搜索过程中对重复分支的访问，在第一次访问结束时对这一次访问的状态做一个标记，当下一次再次来访问相同的分支时，直接使用第一次访问的结果，而不需要重复搜索。
对于记忆化搜索的题，要自然而然得想到动态规划优化，往往都是可行的，但是在理解上需要更多得去琢磨，即如何把记忆化 DFS 转变成 DP
