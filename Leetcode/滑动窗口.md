# 双指针与滑动窗口题型总结
这种题型一般很明显，也是一种贪心的策略，使用双指针来实现。双指针限制了我们讨论元素的首尾边界，中间的元素也会借助很多数据结构来维护（如unordered_map、队列、堆、栈等）。这个专题学到了一种集合的思想：即把固定值转化为两个大集合的差，题992与题930都运用了这种思维，非常得巧妙。
